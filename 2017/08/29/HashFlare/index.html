目前提供的合约:

Bitcoin 永久合约: SHA-256, Scrypt
Ethereum 一年合约: ETHASH
DASH 一年合约: X11

HashFlare注册网址(可能需要翻墙): https://hashflare.io/r/61B0D9A6

通过本链接成功购买的, 请发送 金额 和 日期 到guan_g_h@163.com, 领取3%返现!!!

操作教程:

1: 打开网站点击注册，如下图！

2: 登陆后,点击”设置”填写自己的钱包地址,并且保存,收到确认信.

3: 点击”购买利率”,选择自己想买的合约.选取想要的数量,点击”处理”

4: 点击”选择支付方式”,选择你想使用的付款方式.勾选”我确认已阅读并同意”,并按下确认.


5: 信用卡支付方式
5.1: 选择“使用信用卡付款”作为付款方式。然后，您将被带到LPB托管页面。

请填写所有字段：全卡号，有效期，CVV / CVC安全码。点击“安全付款”。

5.2: 如果您的卡支持3DSecure验证：MasterCard SecureCode或Visa验证，您将被带到您的银行完成付款。您将被要求输入该卡的确认码。

NB！如果卡不支持这些标准，您将看不到此页面。转到步骤3。(图片可能不一样)

5.3: 系统将把您的卡上的全额付款保留。您现在需要付款说明中的4位数安全码。

请在网上查询银行对帐单中的付款说明。有些银行可能会通过短信发送付款说明，也可以通过联系您的银行支持来了解。该声明将显示预留金额和代码将在交易明细中显示（下图所示）。

5.4: 要完成验证，您将被带回HashFlare网站，在相关输入字段中输入付款安全码。点击“保存”。


如果由于某些原因关闭或退出验证屏幕，可以通过单击仪表板中的链接返回到此屏幕。

如果由于某些原因链接不存在，您可以直接输入hashflare.io/panel/verify/XXXXXXX，其中XXXXXXX是您的购买ID（例如hashflare.io/panel/verify/1234567）。

5.5: 恭喜！您的卡现在已被验证在HashFlare.io中使用！

6: 配置你的矿池


7: 收益计算

(目前建议购买SHA-256挖矿合约,每天的收益最高!约为0.4%)

SHA-256,Scrypt合约为永久合约会收取维护费用
Scrypt每一天每1MH/s 收取0.01美元的维护费用
SHA-256 每一天每10GH/s 收取0.0035美元的维护费用

ETHASH,X11 为一年合约,没有维护费用

你可以在这里计算出未来的收益情况.
点击进入收益计算器

附註 : 1 TH/s = 1000 GH/s = 1000 MH/s = 1000000 KH/s

SHA-256 (挖比特币)
请填写你的 Hashing Power (挖矿算力)
Power consumption 跟 Cost per KW/h 都填 0 就好，算出纯收益减掉维护费用就是自己每日的收益
例子 : 如果目前购买 1TH/s的算力 算出每日纯收益为 0.858美元扣除 0.35美元 (0.0035*100)的维护费用，你每天的收益就是 0.508美元

Scrypt (挖莱特币自动换成比特币) 也是一样算出纯收益扣除维护费用
就是自己每日的收益

ETHASH (挖以太币)、X11 (挖达世币)直接填写自己的 Hashing Power (挖矿算力)
Power consumption 跟 Cost per KW/h 都填 0 就好
算出来的收益就是自己每天的收益
